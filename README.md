# Build usage simple Bard API

###### tags: `public`,`build`,`bard`


## How to use?

---
Clone from my github

```
git clone https://github.com/tingruikp0925/bardapi_simple.git
```

Install Python3

```
brew install python3
```

Install requirement
```
pip3 install -r requirements.txt
```

Run it
```
python  bard_api.py
```
